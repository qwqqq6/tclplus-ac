# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0

Initial release.

- Add TCL+ domestic App account login.
- Add automatic token refresh.
- Add AC device discovery.
- Add `climate` entity for power, HVAC mode, temperature, fan mode and swing mode.
- Add switch entities for power, auto fan, soft wind, ECO, PTC, beep, screen, anti-mildew, self learn and self clean.
- Add select entities for sleep mode and wind direction.
- Add number entity for fan speed percentage.
- Add sensors for temperatures, wind state, clean state, current, voltage, compressor frequency and error code.
- Add TCL+ App-like linked control and disabled-control rules.
