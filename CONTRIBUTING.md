# Contributing

感谢你愿意改进 TCL+ AC。

## 反馈兼容性

如果你的设备可以正常使用，欢迎提交：

- 空调型号。
- 产品 key。
- Home Assistant 版本。
- TCL+ App 版本。
- 哪些控制项已验证可用。

请不要公开账号、密码、token、完整设备 ID 或家庭网络信息。

## 提交 bug

提交 bug 时请包含：

- 问题描述。
- 复现步骤。
- 期望结果。
- 实际结果。
- Home Assistant 版本。
- 集成版本。
- 脱敏日志。

## 提交代码

建议流程：

1. Fork 仓库。
2. 创建功能分支。
3. 修改代码和文档。
4. 运行语法检查。
5. 提交 Pull Request。

语法检查示例：

```bash
python -m compileall custom_components/tclplus_ac
```

## 代码风格

- 保持运行时零第三方依赖，除非有明确收益。
- 新控制项应尽量来自真实 App 行为或设备返回数据。
- 不要输出敏感信息到日志。
- 不要提交逆向缓存、抓包、账号、token、设备 ID。
- 文档优先写清楚限制和验证范围。

## Pull Request 清单

- [ ] 代码可以编译。
- [ ] 没有提交敏感信息。
- [ ] 更新了相关文档。
- [ ] 说明了测试设备或测试方式。
- [ ] 如果新增实体，已补充实体说明。
